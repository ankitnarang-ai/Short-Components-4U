Follow these steps to install and configure the Angular Kanban component

01

terminal Install via NPM

Add the Angular Kanban library to your project dependencies

bash 
npm install angular-kanban
02

code Import Component

Import the KanbanBoardComponent in your Angular module or standalone component

typescript 
import {{ '{' }} KanbanBoardComponent {{ '}' }} from 'angular-kanban';
03

web Add to Template

Use the kanban-board component in your Angular template

html 
<lib-kanban-board 
  [board]="board" 
  (cardClick)="onCardClick($event)" 
  (cardMoved)="onCardMoved($event)">
</lib-kanban-board>

settings Component API

Complete reference for all component inputs, outputs, and methods

input Input Properties

Property Type Required Description
board KanbanBoard Yes The complete board data structure containing columns and cards
cardTemplate TemplateRef No Custom template reference for rendering individual cards
disabled boolean No Disables drag and drop functionality when true

output Output Events

Event Payload Type Description
cardMoved CardMoveEvent Emitted when a card is moved between columns or within a column
cardClicked KanbanCard Emitted when a card is clicked or tapped
columnChanged ColumnEvent Emitted when column properties are modified
